= Contributing to Limited Guest Posts =
If you have a patch, or stumbled upon an issue with Limited Guest Posts core, you can contribute this back to the code. Submit your pull request or issue on [Github Limited Guest Posts](https://github.com/mehulkaklotar/limited-guest-posts)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Limited Guest Posts'
3. Activate Limited Guest Posts from your Plugins page.

= From WordPress.org =

1. Download Limited Guest Posts.
2. Upload the 'limited-guest-posts' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Limited Guest Posts from your Plugins page.
